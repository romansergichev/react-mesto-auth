# Фронтенд проекта Mesto на React

**О проекте**

Реализация фронтента проекта Mesto на React.

**Технологии**

HTML, CSS, JavaScrip, Git, Webpack, fetch API, React

**Описание**

Проект перенесён на React. Выполнен полный рефакторинг функционала.

В создании использовались функциональные компоненты и хуки.

**Npm-скрипты**

npm start -  запустить проект на локальном сервере.
npm run build - создать продакшен билд проекта.
